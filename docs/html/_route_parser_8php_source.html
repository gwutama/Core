00051             //
00052             // Strategy:
00053             // 1. Replace words within {} of custom route with regex,
00054             //    move the parameter name/key to an array.
00055             //    Example above: $params = array("param" => null, "param2" => null);
00056             // 2. To check whether $url is actually a custom route:
00057             //    2.1. Replace words of custom route within {} with nothing.
00058             //         Example above: {param} -> {}, {param2} -> {}
00059             //    2.2. Split the custom route based on "{}"
00060             //    2.3. Cross check characters of the split arrays with
00061             //             the actual $url. Note where the first and last positions
00062             //         of the $url.
00063             //    2.4. Based on the first and last positions, fill in the parameter
00064             //         values into $params.
00065             //    2.5. Replace each parameter values of the $url with {variable_name},
00066             //         save it in a temporary variable.
00067             //         Example above: /blah/foo/bar/etc -> /blah/{param}/{param2}/etc
00068             //    2.6. If $tmpVar == $url, build a RoutingObject Object then return it.
00069             //         Otherwise, this loop will ends after the last $route and this
00070             //         method will return null instead.
00071             $params = $this->parseParamsFromRoute($route, $url); // Steps 1 to 2.4
